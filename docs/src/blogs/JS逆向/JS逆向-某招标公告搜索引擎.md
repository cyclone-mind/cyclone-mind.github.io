# JS逆向之招标公告搜索引擎

JS 逆向（JavaScript Reverse Engineering）指通过分析网页中 JavaScript 代码的加密逻辑，还原数据请求的加密参数生成过程。它是爬虫开发中绕过反爬机制的核心技术，常用于处理以下场景：

- 网页通过 eval、webpack 混淆加密参数（如 _signature、token）
- 请求参数包含动态生成的加密字段（如 sign、timestamp）
- 验证码或敏感接口的加密验证逻辑
