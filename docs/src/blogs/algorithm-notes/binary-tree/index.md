## 核心思想

- **递归思想**：二叉树的很多问题都可以通过递归来解决
- **分治思想**：将大问题分解为子问题
- **深度优先搜索（DFS）**：前序、中序、后序遍历
- **广度优先搜索（BFS）**：层序遍历
- **二叉搜索树性质**：中序遍历得到有序序列
